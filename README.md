# leoparx
A lightweight LXMP for CentOS (Apache/Nginx/MySQL/PHP)
Easy to manage server
```
(ROOT)
|--- /leoparx
     |
     |--- /build
     |    |
     |    |--- /sources
     |    |--- /configurations
     |         |--- /php73
     |              |--- /configure.php73
     |              |--- /etc
     |              |    |--- /php-fpm.conf
     |              |--- /logrotate.d
     |              |--- /systemd
     |--- /applications
     |    |--- /nginx
     |--- /data
```
